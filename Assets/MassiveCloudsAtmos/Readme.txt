[ChangeLog]
----------------------------------------

Version 1.1.0 (2021-02-21)

  Feature: HDRI support for Skybox
  Feature: Star HDRI cubemap included
  Fix: Improve performance and quality of Sun Shaft
  Fix[HDRP]: Light intensity is not set properly from AtmosPad

Version 1.0.9 (2020-12-27)

  Feature: HDRP/URP 10.2.2 supported
  Fix[HDRP]: Fix brightness of clouds on Reflection Probe with HDRP Custom Pass 

Version 1.0.8 (2020-10-02)

  Feature[HDRP]: Add option to lookup Transparent Depth Prepass in Custom Pass.
  Fix[Metal]: Shader error on HDRP 7.5.1
  Fix[HDRP]: Incorrect rendering when set MSAA within Forward.
  Fix[URP]: Incorrect rendering when opaque texture not used in pipeline settings.

Version 1.0.7 (2020-09-09)

  Feature: New Sky Renderer
  Feature: Update Presets
  Feature: Add option to call DynamicGI.UpdateEnvironment automatically.
  Feature: Update Cloud Ambient Lighting.
  Fix: Sun disk is too bright and making artifact in Sky Renderer.
  Fix: Separate HDRP shadervariants to avoid problems on build.

Version 1.0.6 (2020-08-23)

  Fix[URP]: Physics Renderer parameter cause null reference exception

Version 1.0.5 (2020-08-20)

  Feature: Sky Parameter Controller in AtmosPad

Version 1.0.4 (2020-07-30)

  Fix[Built-in]: Optimize CameraEffect rendering.

Version 1.0.3 (2020-07-20)

  Fix[URP]: Following camera is broken on Universal RP 7.4 or later.
  Fix: Can not fly through clouds with StylizedRenderer.

Version 1.0.2 (2020-07-11)

  Feature[HDRP]: Enable HDRP Sky Intensity Mode
  Feature[HDRP]: Dynamic Ambient Mode Setting in Setup Wizard
  Feature: Sun disk rendering on Sky module
  Fix: Sky Lighting Updating with HDRP Sky
  Fix: Text color is broken in the Unity Editor Normal Skin.

Version 1.0.1 (2020-07-03)

  Fix: Text is hard to read in the Unity Editor Normal Skin.

Version 1.0.0 (2020-06-17)

  Released.
